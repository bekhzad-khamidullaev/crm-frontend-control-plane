================================================================================
⚠️  ACTION REQUIRED BEFORE DEPLOYMENT:
================================================================================

1. UPDATE SIP CREDENTIALS
   - Edit .env.production
   - Set VITE_SIP_USERNAME and VITE_SIP_PASSWORD

2. SETUP SERVER (First Time Only)
   - Run: sudo ./scripts/setup-server.sh
   - Obtain SSL certificates: sudo certbot --nginx -d windevs.uz

3. CONFIGURE DNS
   - Point windevs.uz → Server IP
   - Point www.windevs.uz → Server IP

4. VERIFY BACKEND
   - Ensure crm.windevs.uz/api/ is accessible
   - Ensure pbx.windevs.uz is running
   - Configure CORS to allow windevs.uz

5. SETUP GITHUB SECRETS (for CI/CD)
   - PRODUCTION_HOST
   - PRODUCTION_USER
   - PRODUCTION_SSH_KEY
