ON YOUR LOCAL MACHINE:
----------------------
1. Install missing npm dependencies:
   npm install --save-dev rollup-plugin-visualizer vite-plugin-compression @vitest/coverage-v8

2. Update SIP credentials in .env.production:
   VITE_SIP_USERNAME=<your-username>
   VITE_SIP_PASSWORD=<your-password>

3. Test production build:
   npm run build:production
   npm run preview:production

4. Commit and push:
   git add .
   git commit -m "Production deployment ready"
   git push origin main

ON PRODUCTION SERVER:
---------------------
5. Configure DNS: windevs.uz → server IP

6. Clone repository:
   git clone <repo-url> /opt/crm-frontend
   cd /opt/crm-frontend

7. Setup server (installs Docker, nginx, certbot, etc.):
   sudo ./scripts/setup-server.sh

8. Get SSL certificates:
   sudo certbot --nginx -d windevs.uz -d www.windevs.uz

9. Deploy:
   ./deploy.sh production

10. Verify:
    ./scripts/health-check.sh
